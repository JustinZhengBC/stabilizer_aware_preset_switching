Purpose

Typically, stabilizer is used for drawing but not erasing. This plugin alters keyboard shortcut behaviour to account for this when switching presets or erase mode.

Usage

After installing the plugin and restarting Krita:
  1. Activate by clicking Tools > Scripts > Stabilizer-Aware Preset Switching
  2. Select an eraser preset, then a pen preset
  3. Use keyboard shortcuts to switch between them
  4. The stabilizer should now turn on/off automatically
  5. This also works for erase mode
  6. Repeat step 1 to disable
